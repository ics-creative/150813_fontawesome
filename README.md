# Font AwesomeをHTML CanvasとWebGPUで使う方法のサンプル

詳しくは[ICS MEDIAの記事](https://ics.media/entry/8385)をご覧ください。

- `index.html`: デモ一覧
- `1_simple.html`: DOMでアイコンフォントを表示
- `2_canvas.html`: Canvasで単体アイコンを描画
- `3_canvas_all.html`: Canvasでアイコン一覧を描画
- `4_threejs.html`: WebGPUで単体アイコンを表示
- `5_threejs_all.html`: WebGPUでアイコン一覧を表示
