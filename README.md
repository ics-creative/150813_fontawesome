# ウェブフォントをHTML CanvasとWebGPUで使う方法のサンプル

詳しくは[ICS MEDIAの記事](https://ics.media/entry/8385)をご覧ください。

- `index.html`: デモ一覧
- `01_dom_japanese.html`: DOMで日本語フォントを表示
- `02_dom_icon.html`: DOMでアイコンフォントを表示
- `11_canvas_japanese.html`: Canvasで日本語フォントを描画
- `12_canvas_icon.html`: Canvasで単体アイコンを描画
- `13_canvas_icon_all.html`: Canvasでアイコン一覧を描画
- `21_webgpu_japanese.html`: WebGPUで日本語フォントを表示
- `22_webgpu_icon.html`: WebGPUで単体アイコンを表示
- `23_webgpu_icon_all.html`: WebGPUでアイコン一覧を表示
